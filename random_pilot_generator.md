# Random Pilot Generator

Created a small Python script that generates a random pilot name, agency name, and aircraft number.

## Modification log
- Added `random_pilot_generator.py` at the project root.
- The script uses Python's `random.choice()` to pick values from predefined lists.
- The script prints one random combination in a card-style details block when run directly.
- The checkout details card now also shows pilot, agency, and avion values in the Symfony UI.
- Reduced checkout details modal spacing and enabled proper modal-body scrolling so bottom action buttons remain visible.

## Run it
```bash
python random_pilot_generator.py
```
